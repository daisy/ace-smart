# Ace SMART

Public repository for the Ace SMART tool.

The Ace SMART source code is currently not publicly available.

## Feedback

This repository is used to track issues with Ace SMART. To report a bug, request a feature, or provide feedback, please open a new issue in the [issue tracker](issues).

Before opening a new issue, please search the issue tracker for possible matches.

You will need to create a GitHub account in order to add issues.

## User Guide

For a guide to using the Ace SMART tool, refer to the [Ace SMART user guide](http://smart.daisy.org/user-guide/).

## Wiki

This repository also contains the [public wiki](/wiki) for Ace SMART.
